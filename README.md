# UFinAKA Prototype

This project implements a simplified end-to-end prototype of the
UFinAKA fingerprint-based authentication and key agreement protocol.

## Assumptions

- Fingerprints are simulated strings
- Cryptography uses hash, XOR, and random nonces
- No plaintext password or biometric is stored

## Features

- Registration
- Mutual authentication
- Session key agreement
- Blind credential update
- Adaptive biometric threshold (to be added)

This prototype is designed for academic demonstration.
